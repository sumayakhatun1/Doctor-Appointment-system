# Doctor Appointment System (Frontend)

Clean and simple **semester project UI** using:

- React (Vite)
- React Router DOM
- Tailwind CSS
- Dummy data only (no backend)

## Folder Structure

```
src/
  components/
  layouts/
  pages/
  routes/
  data/
```

## Demo Accounts

- Patient: `sara@demo.com` / `patient123`

## Run Locally

```bash
npm install
npm run dev
```

## Notes (Academic-Friendly)

- Authentication is **state-based** using `React Context + localStorage`.
- Appointments are stored in `localStorage` (so booking/status updates look realistic).
